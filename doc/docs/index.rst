bde_verify
==========

.. only:: bde_verify
   
   `bde_verify <https://github.com/bloomberg/bde_verify>`_ is a
   tool that performs a variety of checks on the C++ source code for adherence
   to ``BDE`` design rules, coding standards and detects potential errors.
